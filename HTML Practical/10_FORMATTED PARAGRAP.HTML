HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects such as interactive forms may be embedded into the rendered page HTML provides a means to create structured documents by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes and other iiems.
HTML elements are delineated by tags, writen using angle brackets. Tags such as< img > and <input/> direeily introduce content into the page.
Other tags such as <p> surround and provide infomation about document text and may include other tags as sub-elements.
Browsers do not display the HTML tags but use them to interpret the content of the page.
